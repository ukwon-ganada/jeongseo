# 기일연기·변경신청서 (클로드 사유작성 + HWPX 다운로드) 설정 가이드

정서 웹 서면작성에 **기일연기·변경신청서** 기능을 추가했습니다.
직원이 간단히 사유를 메모하면 **클로드가 정서 문체로 사유 문단을 작성**하고,
완성된 서면을 **한글(.hwpx) 파일로 다운로드**합니다.

---

## 추가·변경된 파일

| 파일 | 내용 |
|---|---|
| `yeongi.js` | 프론트 모듈 — 입력폼·클로드 호출·HWPX 생성(JSZip)·다운로드 |
| `supabase/functions/draft-yeongi/index.ts` | Edge Function — 클로드로 사유 문단 작성(API 키는 서버 시크릿) |
| `templates/yeongi_criminal.hwpx` | 형사 표준양식(정서 원본) |
| `templates/yeongi_civil_consent.hwpx` | 민사 동의 표준양식 |
| `templates/yeongi_civil_dissent.hwpx` | 민사 부동의 표준양식 |
| `index.html` | `<script src="./yeongi.js">` 추가 |
| `home.js` | 홈 메뉴 `기일연기·변경` 항목 추가 |

> HWPX 생성은 **브라우저에서** JSZip으로 처리하므로(시크릿 불필요), 배포에서 챙길 것은 아래 **Edge Function(클로드) 하나뿐**입니다.

---

## 설정 순서

### 1) Anthropic API 키를 Supabase 시크릿으로 저장
API 키는 **브라우저에 노출되지 않고 Edge Function 안에서만** 사용됩니다(솔라피 방식과 동일).

```bash
supabase secrets set ANTHROPIC_API_KEY=sk-ant-...
# (선택) 모델 변경 — 기본값 claude-opus-4-8
supabase secrets set YEONGI_MODEL=claude-opus-4-8
```

또는 Supabase 대시보드 → Edge Functions → Secrets 에서 `ANTHROPIC_API_KEY` 추가.

### 2) Edge Function 배포
```bash
supabase functions deploy draft-yeongi --no-verify-jwt
```
`--no-verify-jwt` 는 프론트가 `apikey` 헤더로 호출하기 때문입니다(otp-send 와 동일).

### 3) 프론트 배포
`yeongi.js` · `templates/` · 수정된 `index.html`·`home.js` 를 평소대로 배포하면 끝.
홈에 **「기일연기·변경」** 카드가 나타납니다.

---

## 사용 흐름

1. 홈 → **기일연기·변경**
2. **사건 구분**(형사/민사) 선택 → 사건번호 입력 시 자동완성(기존 `autofill.js`)
3. 기일·희망기일·담당변호사·첨부 입력
4. **사유 메모**에 간단히 적고 → **✨ 클로드로 사유 작성** → 정서 문체 사유가 채워짐(검토·수정 가능)
5. **📄 HWPX 다운로드** → 완성된 `.hwpx` 저장 → 한글에서 열어 확인·제출

---

## 구현된 규칙 (엔진)

- **제목**: 기일종류(공판/변론/선고/없음) × 동작(연기/변경) 조합
- **형사**: 피고인/복수 · 국선(‘법무법인 정서’ 줄 생략 + ‘국선변호인’) · 담당변호사 복수 나열 · 첨부 유무
- **민사 동의**: 원고→피고 양측 소송대리인 · ‘위 동의함’ · 위 동의함부터 2페이지(우리가 원고면 상대방 블록 위로 이동)
- **민사 부동의 / 대리인부존재**: 우리측 소송대리인만 · 부동의/대리인부존재 문구 자동 · ‘변경신청사유 :’ 접두어
- **A4 자동맞춤 v4**: 220%→190% 축소 시도 → 둘 다 초과 시 **220% 복원 + 단일 페이지나눔**(첨부 세트 보존, 3페이지 금지, 2페이지 완결)
- **본문 첫줄 들여쓰기** 보정(형사)

## 주의 (법률 도메인)

- 클로드는 **사유 초안**만 작성합니다. **반드시 변호사가 검토·수정 후 제출**하세요.
- 시스템 프롬프트에 “사용자가 준 사실만 사용, 없는 사실·법조문·판례·사건번호 창작 금지”를 명시했지만, 최종 확인 책임은 작성자에게 있습니다.
- 서면 서식(글꼴·여백·표)은 정서 원본 `.hwpx` 템플릿을 그대로 보존하고 **텍스트만** 채웁니다.
